Änderungen
Folgende Änderungen sind durchzuführen:

1.) Heruntergeladene Datei (auswert_fuer_webmaster.rar) entpacken

2.) Dateien auf Server kopieren
\auswert_admin.php -> ins LMO-Root
\auswert2.php -> ins LMO-Root
\auswert_funk.php -> ins LMO-Root
\auswert.cfg -> ins LMO-Root

3.) Zugriffsrecht auf Server manuell ändern
\auswert.cfg -> chmod 666

Infos zu chmod bei Wikipedia.
Anleitung dazu mit dem Freeware-Tool Filezilla.

4.) "auswert_admin.php" im Browser aufrufen - Bsp.: www.domain.de/auswert_admin.php
5.) Einstellungen eingeben, anschließend speichern

Das fertige Resultat kann nun betrachtet werden - Bsp: www.domain.de/auswert2.php

Hinweis: Die Datei "auswert_admin.php" nun am besten vom Server löschen!

Die Auswertung kann nun in die eigene Homepage mittels
include("auswert2.php");
eingebunden werden.

Hinweis
Info: wer das Skript einsetzt, muss jeweils zur neuen Saison die "auswert_admin.php" neu ausführen.
(Skript auf Server laden - ausführen - vom Server löschen)

Grund: die neuen Ligen für die neue Saison müssen neu in die Config-Datei geschrieben werden. Sonst kann es sehr wahrscheinlich zu falschen Werten kommen, da "falsche" Ligen zusammenaddiert werden.
